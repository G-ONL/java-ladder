# java-ladder
사다리타기 미션 저장소

## 기능과 예외상황
(기능)
  1. 커스텀 구분자 지정

  	 "//;\n" -> ";"

     (예외)
     1. 커스텀 지정자 하나 이상
     2. 커스텀 지정자에 숫자가 들어가는 경우

  2. 구분자를 기준으로 Split

     "1,2,3" -> "[1,2,3]"
     "1:2:3" -> "[1,2,3]"
     "1,2:3" -> "[1,2,3]"
     "1;2;3" -> "[1,2,3]"

     (예외)
     1. 구분자가 연속되는 경우
     2. 숫자가 연속되는 경우
     3. 구분자 리스트외에 값이 들어온 경우
     
  3. 숫자의 합을 반환

     "[1,2]" -> "3"
     "[1,2,3,4]" -> "10"

     (예외)
     1. 숫자 이외의 값 또는 음수
